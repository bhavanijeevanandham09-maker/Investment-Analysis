This project analyzes investment data to identify trends, risks, and returns. 
The goal is to help make better financial decisions using data.
## Dataset
The dataset contains investment details such as returns, risk levels, and categories.
## Analysis Performed
- Cleaned and organized the data
- Analyzed investment performance
- Created visual dashboards
- Identified key trends and insights
- ## Key Insights
- High-risk investments gave higher returns
- Some categories showed stable growth
- Seasonal trends were observed
- ## Tools Used
- Excel
- Power BI.

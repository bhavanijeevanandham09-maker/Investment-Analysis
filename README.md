# Investment Analysis Project

## About the Project
This project analyzes investment patterns based on gender and behavior. 
It focuses on understanding how people invest and identifying trends across different investment options.

## Tools Used
- Excel
- Power BI

## Dataset
The dataset includes information about individuals, their gender, and their investment choices such as gold, mutual funds, and equity.

## Analysis Performed
- Cleaned and structured the dataset
- Analyzed investment preferences based on gender
- Compared different investment options
- Created visual dashboards for better understanding

## Key Insights
- Investment choices vary significantly based on gender
- Gold and mutual funds are popular among certain groups
- Equity investments show different risk-taking behavior
- Clear patterns observed in how people diversify their investments

## Conclusion
This analysis helps understand investor behavior and can support better financial decision-making.
